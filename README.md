# Formation — BPMN, UML & Study Case

Support de la formation de 3 jours combinant **BPMN**, **UML** et un **Study Case**.

## 📂 Contenu du dépôt

| Dossier       | Contenu                                                       |
| ------------- | ------------------------------------------------------------- |
| `BPMN/`       | Pages théoriques sur la notation BPMN                         |
| `UML/`        | Pages théoriques sur les diagrammes UML                       |
| `Study Case/` | Étude de cas progressive — énoncés et solutions des exercices |

Les fichiers sont au format **Markdown** et se consultent directement depuis l'interface GitHub : il suffit de cliquer sur un fichier `.md` pour l'afficher avec le formatage, les diagrammes et les liens entre pages.

## 🔄 Mises à jour pendant la formation

Le contenu est publié **au fur et à mesure** des trois jours :

- Les supports théoriques sont ajoutés avant chaque module.
- Les énoncés d'exercices sont publiés au début de chaque session pratique.
- Les solutions et améliorations sont poussées après la correction en groupe.

Rafraîchissez simplement la page GitHub pour voir les dernières mises à jour.

## ⬇️ Télécharger l'intégralité du dépôt

À la fin de la formation, vous pouvez récupérer l'ensemble du contenu de deux manières :

**Option 1 — Télécharger une archive ZIP**
Cliquer sur le bouton vert **`Code`** en haut de la page, puis **`Download ZIP`**.

**Option 2 — Cloner avec Git**

```bash
git clone <url-du-depot>.git
```
